This is the first submission of Assignment 2.

Source code is in file 3Dgame.cpp
How to start- 

1. run `make` command on the terminal
2. Then run the executable `./game` 

This is just a basic version of the game.

The player is a rectangular white block.

The main objective of the game is to collect the red flag.

Obstacles are in the form of holes.
If you fall into a hole, it will automatically reset the players position.

3 camera views have been implemented:
1. Tower View
2. Side View
3. Top View

The camera view can be changed by pressing the 'V' key on the keyboard.
The player can be moved using arrow keys.
Zoom on and zoom out using w and s keys.
